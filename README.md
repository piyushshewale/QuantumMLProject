# Quantum Machine Learning using Qiskit

## Overview

This project demonstrates a simple Quantum Machine Learning (QML) model using Variational Quantum Circuits (VQC) built with Qiskit.

The model performs binary classification on a dataset by combining concepts of:
- Quantum Computing
- Machine Learning
- Quantum Physics
- Data Classification

This project was created for academic demonstration purposes for a Physics / Engineering course project.

# Quantum Concepts Used

## Qubits
Quantum bits capable of existing in multiple states simultaneously.

## Superposition
Allows qubits to represent both 0 and 1 at the same time.

## Entanglement
Creates correlations between qubits for advanced computation.

## Quantum Gates
Operations that manipulate quantum states.

## Variational Quantum Circuit (VQC)
A trainable quantum circuit used for machine learning tasks.

## Quantum Feature Mapping
Converts classical data into quantum states.
